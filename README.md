# Info-Klausur

Das Projekt soll im Rahmen des Informatik Modules an der TH Bingen als Prüfung erstellt werden.
Ziel des Projektes ist, eine Verkehrsimulation zu erstellen. Dieses Ziel soll mit Hilfe der 
folgenden Programme und Server erreicht reicht werden: 
Git, Github, Travis, Maven, Kotlin (IntelliJ IDEA).

Der Dozent unter dessen Übewachung das Projekt zu bearbeiten ist, ist Nicolai Parlog.

Abgabe der Projektarbeit ist am Sonntag dem 13.08.2017 um 18.00 Uhr MESZ.

Gruppenteilnehmer sind:
Sven Lang, Artur Hermann, Patrick Freudenberger, Pascal Oppenhäuser



## Applikation

Um die Applikation auszuführen muss man auf seinem Rechner zunächst Windows Powershell öffnen.
Darüber navigiert man sich in den Ordner, in dem die .jar Datei gespeichert ist (target Ordner).
Jetzt ist der Befehl "java -jar InformatikKlausur-1.0-SNAPSHOT-jar-with-dependencies.jar" einzugeben.
